# debitAccountStep - Medusa Core Workflows Reference

This documentation provides a reference to the `debitAccountStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step debits one or more store credit accounts by creating debit transactions.
The step supports rollback by deleting the created transactions.

## Example

```ts
const data = debitAccountStep([
  {
    account_id: "sca_123",
    amount: 50,
    reference: "cart",
    reference_id: "cart_123",
  },
])
```
