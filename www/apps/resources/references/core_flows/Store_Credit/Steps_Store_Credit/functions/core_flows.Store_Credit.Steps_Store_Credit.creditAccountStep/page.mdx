# creditAccountStep - Medusa Core Workflows Reference

This documentation provides a reference to the `creditAccountStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step credits one or more store credit accounts with transactions.
The step supports rollback by deleting the created transactions.

## Example

```ts
const data = creditAccountStep([
  {
    account_id: "sca_123",
    amount: 100,
    reference: "gift_card",
    reference_id: "gc_123",
  },
])
```
