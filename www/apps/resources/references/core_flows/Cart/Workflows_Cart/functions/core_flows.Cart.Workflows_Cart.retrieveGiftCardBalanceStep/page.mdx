# retrieveGiftCardBalanceStep - Medusa Core Workflows Reference

This documentation provides a reference to the `retrieveGiftCardBalanceStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step retrieves the balance of a single gift card from its associated store
credit account. Returns a map of the gift card code to its account stats.

## Example

```ts
const data = retrieveGiftCardBalanceStep({
  storeCreditAccount: {
    id: "sca_123",
    balance: 100,
    // other store credit account properties...
  },
  giftCard: {
    id: "gc_123",
    code: "GC-XXXX",
    // other gift card properties...
  },
})
```
