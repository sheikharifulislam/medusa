# OrderShippingMethodAdjustment - Order Module Data Models Reference

This documentation provides a reference to the OrderShippingMethodAdjustment data model. It belongs to the Order Module.

The Order Shipping Method Adjustment data model. This model represents adjustments applied to 
order shipping methods, such as promotions or discounts.

:::note

This is available starting from [Medusa v2.13.7](https://github.com/medusajs/medusa/releases/tag/v2.13.7).
